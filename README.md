# Python Project in IBM Data Science Professional Certificate (Course 5 of 12).

## Overview
This project was completed as part of the IBM Data Science Professional Certificate. 
The goal was to extract financial data from various sources using Python Libraries and web scraping techniques. Later, I visualized the data to identify patterns and trends.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Project Steps
1. Use yfinance to Extract Stock Data
2. Use Webscraping to Extract Tesla Revenue Data
3. Use Webscraping to Extract GME Revenue Data
4. Plot Tesla Stock Graph
5. Plot GameStop Stock Graph
